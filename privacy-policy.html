Browser EQ Privacy Policy

This Privacy Policy explains how Browser EQ handles information when you use the extension.

Last updated: April 2026

1. Overview

Browser EQ is a browser extension that provides visual parametric EQ control for supported browser tabs. Its purpose is to let users adjust audio locally inside the browser by editing an equalizer curve, applying built-in presets, and changing output gain.

Browser EQ is designed as a local audio-processing tool. It is not designed for advertising, user profiling, unrelated tracking, or the resale of user data.

2. Information Browser EQ Handles

In order to provide its core equalizer functionality, Browser EQ may handle the following categories of information:

  • Supported tab audio content: Browser EQ processes the audio stream of a supported browser tab after the user explicitly interacts with the extension.
  • Current tab information: Browser EQ may access limited information about the active tab, such as whether the tab is supported and which tab the user is currently working with, in order to apply the selected audio settings correctly.
  • User settings: Browser EQ stores the user’s EQ configuration, selected preset state, and output gain values so those settings can be restored later.

Browser EQ does not ask users to create an account and does not intentionally collect personal identity information such as name, postal address, phone number, or government identifiers.

3. How Browser EQ Uses Information

Browser EQ uses the information above only to:

  • identify the current supported tab,
  • capture and process supported tab audio locally,
  • apply EQ, preset, and output gain settings selected by the user,
  • restore saved settings between sessions.

Browser EQ does not use handled information for advertising, behavioral profiling, creditworthiness decisions, or unrelated analytics.

4. Local Processing and Storage

Browser EQ is intended to process supported tab audio locally in the browser. User settings are stored locally through the browser extension storage system so the extension can preserve the selected EQ profile and related controls.

Browser EQ is not designed to upload tab audio, page content, or saved EQ settings to a remote server as part of its normal operation.

5. Permissions Used by Browser EQ

  • storage — used to save user settings such as EQ curve data, preset selection, and output gain values.
  • tabs — used to identify and manage the current tab so the extension can determine where to apply the selected audio settings.
  • tabCapture — used to access the audio stream of a supported browser tab after user interaction, so the extension can apply equalizer processing.
  • offscreen — used to support background audio processing without opening a visible extra page for the user.

6. Sharing, Selling, and Transfers

Browser EQ does not sell user data.

Browser EQ is not designed to transfer handled information to data brokers, advertisers, or unrelated third parties.

If a future version of Browser EQ introduces remote services, telemetry, cloud sync, or any other external data transfer, this Privacy Policy will be updated before those changes are published.

7. Data Retention

Browser EQ retains saved settings only for as long as they remain in the user’s browser extension storage or until the user removes them by resetting settings, uninstalling the extension, or clearing browser extension data.

8. Security

Browser EQ is designed to minimize data handling and keep processing local wherever possible. However, no software or local environment can be guaranteed to be completely free from security risks. Users remain responsible for the security of their browser, device, and operating environment.
